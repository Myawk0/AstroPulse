# AstroPulse

> Your space companion — daily astronomy photos, 
> real-time launch tracking, and astronauts in orbit.

## Stack
- SwiftUI · async/await · Clean Architecture
- SwiftData (offline cache)
- App Intents — Siri shortcuts
- GitHub Actions CI

## Data Sources
- NASA APOD API — Astronomy Picture of the Day
- Launch Library 2 (The Space Devs) — launches & astronauts

## Features
- [ ] APOD — daily photo + 30-day archive, offline-first
- [ ] Launches — upcoming & past, all agencies (SpaceX · NASA · ESA · ...)
- [ ] Astronauts — who's in space right now, mission history
- [ ] Siri: "Show today's space photo" / "Who's in space?"
